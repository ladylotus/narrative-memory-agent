Narrative Memory Agent — Architecture

Qwen Cloud Global AI Hackathon • Track 1: MemoryAgent • 2026

Tech Stack

  • • Frontend: Next.js (TypeScript)
  • • Backend: FastAPI (Python 3.11)
  • • LLM: Qwen 3.6-flash via OpenAI SDK
  • • Vector DB: ChromaDB (local)
  • • Structured: SQLite (episodic events)
  • • Embedding: Qwen text-embedding-v3

Dual Circuit Design

  • • Circuit A: Creative generation (character voice)
  • • Circuit B: Multi-factor OOC validation
  • • OOC Formula: 5 factors (T/B/D/C/P)
  • • P factor prevents "safe but boring" bias
  • • GenBias: EMA learning from user feedback

Two-Layer Memory + Schema Store

  • • Layer 1: Working Memory (session buffer, persisted)
  • • Layer 2: Episodic (Zwaan 5-dim indexed events)
  • • Character Schema Store (SQLite: traits, arc, relations)
  • • Vector Store (ChromaDB: embedding search for OOC D-score)
  • • Sleep Cycle: facts→integration→pruning
  • • Psychology-backed design (8 cited papers)
