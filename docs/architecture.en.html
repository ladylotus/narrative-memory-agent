Tech Stack

  • Frontend — Next.js 16, TypeScript
  • Backend — FastAPI, Python 3.11
  • LLM — qwen3.6-flash via OpenAI SDK
  • Embeddings — text-embedding-v3
  • Storage — SQLite + ChromaDB
  • Deploy — Docker Compose on Alibaba Cloud ECS

Dual Circuit Engine

  • Circuit A generates 2–5 in-voice options
  • Circuit B scores each on five OOC factors
  • D factor uses real embedding distance, not LLM self-report
  • P factor prevents "safe but boring" collapse
  • GenBias — EMA learning from user feedback

Memory Design

  • Layer 1 — Working Memory, persisted per character
  • Layer 2 — Episodic events, Zwaan 5-dim indexed
  • Schema Store — traits, arc, relationships
  • Sleep cycle — NREM · REM · self-audit → schema updates
  • Grounded in 8 cited cognitive-psychology papers
