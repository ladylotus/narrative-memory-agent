技术栈

  • • 前端：Next.js（TypeScript）
  • • 后端：FastAPI（Python 3.11）
  • • LLM：Qwen 3.6-flash（OpenAI SDK）
  • • 向量库：ChromaDB（本地文件）
  • • 结构化：SQLite（情景事件）
  • • 嵌入：Qwen text-embedding-v3

双回路设计

  • • 回路A：创意生成（角色语气）
  • • 回路B：多因子OOC一致性校验
  • • OOC公式：5因子 (T/B/D/C/P)
  • • P（惊奇度）防止"安全但无聊"偏向
  • • 用户选择 → 反馈写入 → 记忆更新

两层记忆 + 图式存储

  • • 第1层：工作记忆（会话缓冲区，持久化）
  • • 第2层：情景记忆（五维索引事件）
  • • 角色认知图式存储（SQLite：特质、弧光、关系）
  • • 向量存储（ChromaDB：OOC D值向量检索）
  • • 睡眠巩固：事实→整合→剪枝
  • • 心理学支撑：8篇文献引用
